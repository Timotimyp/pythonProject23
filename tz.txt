Помощник-информатор
Telegram bot будет иметь четыре кнопки:
«Цифра»- эта кнопка будет выдавать полную информацию насчёт введённой цифры(квадрат, корень, разные системы исчисления, и тд)- 12шт
«Слово»- эта кнопка будет отвечать за выдачу полной информации по поводу этого слова, такие как морфологически и морфемный разбор и история происхождения слова ударения(библиотека + API)
«Фильм» - эта кнопка будет создавать API запрос на сайт википедия и получать информацию, такую как режиссёр, дата производства, бюджет, кассовые сборы, ссылка трейлер, средний балл оценок на, страна происхождения, например, Кинопоиске(API+обложка фильма)
3.1) Рекомендация фильма по жанру, по году, по среднему баллу оценок(алгоритм)
3.2) Оценка предложенного фильма(Id_user, фильм, жанр, балл, версия фильма)
3.3) Последний просмотренный фильм
3.4) Посмотреть понравившиеся фильмы(балл >)
3.5) Будет ли следующая часть?
4)Покерные комбинации с подробной картинкой.
Также будет доступна кнопка «Добавить в избранное» главной функцией которой будет добавление последнего запроса в некий список, который в свою очередь в любой момент можно будет посмотреть.
5)Cтраница сайта в которой пользователь регестрируется и видет список фильмов которые он смотрел и которые он добавил в закладки и случайный анекдот + там есть ссылка на telegram
